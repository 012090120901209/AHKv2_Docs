# Help Settings

Specify the settings the help should be started with by default. These
settings are permanent; this is, they apply for all current and future
visits to this help.

  ----------------------------- ----------------------------------------------------
  Content font size:            Very small Small Medium (default) Large Very large
  Selected tab:                 Content (default) Index Search
  Sidebar visibility:           Hidden Visible (default)
  Color theme:                  Light (default) Dark
  Quick reference visibility:   Uncollapsed (default) Collapsed
  ----------------------------- ----------------------------------------------------
